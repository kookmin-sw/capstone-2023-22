## :raised_hands: 협업 규칙

### :ok_woman: commit 규칙

commit 메시지는 다음과 같은 양식으로 작성해주세요!

```
${add, feat, refactor, fix}: ${commit 설명}
```

### :raising_hand: Pull Request 규칙

Pull Request는 자유롭게 내용을 요약해서 보내시면 됩니다.
다만, Pull Request의 제목을 읽고 무슨 내용인지 명확하게 이해할 수 있어야 합니다.
